本地同步说明

此方法使用浏览器的localStorage和IndexedDB实现数据同步。所有更改会同步到同一浏览器的其他标签页。

特点：不需要网络连接，数据不会离开您的设备。

GitHub Gist同步说明

此方法使用GitHub Gist存储数据，通过GitHub账户实现跨设备同步。

需要GitHub授权才能使用此功能

注意：您需要一个GitHub账户，数据将存储在您的GitHub Gist中。

P2P同步说明

此方法使用WebRTC技术实现点对点直接连接，无需中央服务器。

连接设置

使用步骤：

  1. 复制你的ID并发送给对方
  2. 在对方设备上输入你的ID并连接
  3. 建立连接后数据将自动同步
